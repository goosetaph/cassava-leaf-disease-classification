STEPS TO RUN THE WEBSITE LOCALLY

1. Unzip the rar file
2. Go into the folder and create new Virtual Environment (open terminal in the folder and type "python -m venv tubesviskomenv"
3. Activate the Virtual Environment (open terminal in the folder and type "tubesviskomenv\Scripts\activate")
4. install the necessary libraries (run "pip install Flask tensorflow numpy Pillow" on the terminal)
5. Run the Python app ("python app.py" in the terminal)
6. Go to a web browser and go to "http://127.0.0.1:5000"

P.S.: the model is not available here due to its size being 1.2GB
